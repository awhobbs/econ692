## Beamer Theme & Environments

**Preamble:** `Preambles/header.tex` -- 16:9, 12pt, XeLaTeX, Helvetica Neue / Menlo

**Colors:** `usfgreen` (#00543C), `usfgold` (#FDBB30), `darkgray` (#333333), `codebg` (#F0F0F0)

**Macros:** `\E` (expectation), `\Var`, `\indep` (independence symbol)

**Section slides:** Auto-generated by `\AtBeginSection[]` -- just use `\section{}`

| Element | Syntax | Effect | Use Case |
|---------|--------|--------|----------|
| Alert block | `\begin{block}{Title}` | Green header + gray body | Reminders, key definitions |
| Two columns | `\begin{columns}[T]` | Side-by-side layout | Diagram + text |
| Terminal code | `\lstlisting[style=terminal]` | Gray box, bash syntax | Shell commands |
| Green bold | `\textbf{\textcolor{usfgreen}{...}}` | USF green bold | Emphasis in prose |
| Plain title | `\begin{frame}[plain]` | No frame chrome | Title slide only |
